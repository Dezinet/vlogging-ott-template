# 🗺️ Project Roadmap (2026)

This roadmap outlines the planned evolution of **DezinetVerse**.

## 📍 Short-Term (Current Focus)

-   **[ ] Mobile UX Polish:** Optimize hamburger menu height/fonts and hide scrollbars. *(In Progress)*
-   **[ ] Core Architecture Docs:** Add diagrams and technical summaries. *(In Progress)*
-   **[ ] Custom Video Player:** Integrate "Ad Logic" placeholders and sleek controls. *(Done)*

## 📍 Mid-Term (Next 3 Months)

-   **[ ] Skeleton Loaders:** Page-level and component-level loaders to replace full-screen `loading.tsx`.
-   **[ ] Interactive Gear Page:** Hotspot images for the gear section with detailed popovers.
-   **[ ] Better Search UI:** Predictive search in the Discovery Dashboard as the user types.

## 📍 Long-Term (Phase 2)

-   **[ ] Multi-Creator Support:** Dashboard for multiple vloggers under one umbrella.
-   **[ ] Live Events System:** Premiere counting system for new vlog releases.
-   **[ ] Contentful/Sanity Integration:** Move static video data into a headless CMS for better management.
-   **[ ] picture-in-picture (PiP):** Floating player that persists across internal page navigation.

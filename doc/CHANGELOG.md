# 🔄 Changelog

All notable changes to the DezinetVerse project will be documented here.

## [0.2.0] - 2026-04-07

### 🚀 Added
-   **Custom Video Player:** Implemented a bespoke player in `CustomPlayer.tsx` with ad injection and skip logic.
-   **Documentation Hub:** Created the `/doc` folder with Architecture, Roadmap, Changelog, and Theme guides.
-   **Mermaid Diagrams:** Detailed visual diagrams for architecture and theme hierarchy.

### 🎨 Fixed/Improved
-   **Scrollbar Design:** Styled global scrollbars to match the premium theme (Light: Indigo, Dark: Violet).
-   **Mobile UX:** Reduced height and font sizes of the hamburger menu for better accessibility.
-   **Clean UI:** Hidden scrollbars in the Discovery Dashboard using `.no-scrollbar` utility.
-   **Text Updates:** Replaced default ad text with "implement your ad logic here..".

## [0.1.0] - Initial Release
-   Initial Premium Vlogging template with Hero Slider and Series support.

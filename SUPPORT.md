# 🙋 Support Policy

Need help with Vlogging/OTT? We have several ways to get the support you need.

## 🐛 Found a Bug?

Please report bugs using our [Issue Tracker](https://github.com/your-repo/vlogging_template/issues). Before opening a new issue, please:

1.  **Search existing issues** to see if it's already being discussed.
2.  Provide a **Reproducible Example** (e.g., a small code snippet or repo link).
3.  Include your **Environment Details** (Browser version, OS, etc.).
4.  Specify what **version** of the project you are using.

## ✨ Feature Requests

We love suggestions! You can open a feature request on the issues page using our [Template](.github/ISSUE_TEMPLATE.md).
